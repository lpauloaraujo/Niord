# This project uses UV

Make sure to install uv <br> https://docs.astral.sh/uv/getting-started/installation/

To run the enviroment use the following commands.

To install all dependencies:
```bash
uv sync
```

_(Make sure you are in the ./server folder, the root folder of this README)_<br>
To run the fastapi in development mode: 
```bash
uv run fastapi dev
```
