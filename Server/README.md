# How to setup the enviroment


**This Project uses UV**<br>
Make sure to install uv <br> https://docs.astral.sh/uv/getting-started/installation/

## Running in development
_(Make sure your working directory is the ./server folder, the root folder of this README)_

To run the enviroment in development do the following<br>

1. Create a .env file containing the variables read at the file _./src/config.py_ in the _Settings_ class
2. Install all dependencies with:
  ```bash
  uv sync
  ```
3. Run fastapi in development mode with uv: 
  ```bash
  uv run fastapi dev
  ```

## Building the docker image and running
_(Make sure your working directory is the ./server folder, the root folder of this README)_

Make sure to have Docker installed <br>
https://www.docker.com/

1. Build the docker image with:
   ```bash
   docker build --no-cache -t niord ./
   ```
2. Open the Docker Desktop and run the image that should be named _niord_.
3. Define the enviroment variables(equivalent of .env) on the optional settings and define the port to 80.
4. Run the image and check the logs for possible error.

If everything went right you should be able to access the server through _localhost_ with the port especified (port 80 in this case).

## Final considerations
FastAPI have an autogenerated /docs page (Ex. localhost/docs) that shows all the endpoints, allowing direct manual testing of endpoints and knowledge of it's expected parameters.
